# Tài liệu tham khảo

## 1. Mã hóa
- **XOR Cipher:** Phép toán XOR bit cơ bản, mã hóa đối xứng đơn giản nhất
- **AES-128-CBC:** Advanced Encryption Standard - Cipher Block Chaining mode
- **NIST SP 800-38A:** Recommendation for Block Cipher Modes of Operation

## 2. Phần cứng
- **ESP32 Datasheet:** Espressif Systems, dual-core Xtensa LX6, built-in AES accelerator
- **LoRa SX1278:** Semtech, 433 MHz, long-range spread spectrum communication
- **GPS NEO-6M:** u-blox, 50 channels, -161 dBm sensitivity

## 3. Thư viện
- **Mbed TLS:** Thư viện mật mã cho thiết bị nhúng, có sẵn trên ESP32
- **PyCryptodome:** Thư viện mật mã Python
- **Flask:** Micro web framework Python
- **Streamlit:** Dashboard framework Python
- **Folium + Leaflet.js:** Thư viện bản đồ OpenStreetMap

## 4. Tài liệu đồ án
- File hướng dẫn mã hóa: `1-Mahoa.docx` và `2-Mahoa2.docx`
- Code mẫu XOR: `server/xor_cipher.py`
- Code mẫu AES-CBC: `server/app.py`, `wokwi/sketch.ino`
